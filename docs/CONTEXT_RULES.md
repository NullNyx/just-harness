# Context Rules

## Thứ tự đọc

1. `CONTEXT.md`
2. `README.md`
3. `docs/HARNESS.md`
4. `docs/FEATURE_INTAKE.md`
5. `docs/profile/` hoặc profile docs tương ứng, nếu có
6. `docs/notes/INDEX.md` và active notes liên quan, nếu có
7. product docs liên quan
8. decision docs liên quan

## Luật chọn lọc

Chỉ đọc thứ cần cho task. Không nạp cả repo vào prompt nếu task nhỏ.

## Luật evidence

Mọi kết luận phải gắn với file, lệnh, hoặc artifact cụ thể.
