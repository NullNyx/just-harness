# Hồ sơ dự án

Profile điều chỉnh harness behavior cho từng repo.

## Các profile sẵn

- `backend`
- `frontend`
- `full-stack`
- `cli`
- `docs-only`
- `custom`

## Nguyên tắc

Profile nên define:

- phase nào bật
- proof commands nào bắt buộc
- skill hints nào nên dùng
- context order tweaks nào cần đổi

Profile không nên hardcode truth riêng của app; phần đó thuộc CONTEXT.md hoặc docs/notes/.
